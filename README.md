# 📊 Customer Churn Analysis Dashboard

## 🌍 Overview
This project explores customer churn behavior using data analysis and visualization techniques. The goal is to identify patterns that influence customer retention and provide insights that support better decision-making 🧠📉

The dashboard highlights key performance indicators (KPIs) such as churn rate, total customers, and relationships between customer attributes and churn.

---

## 🎯 Objectives
- 🔍 Analyse customer churn trends  
- 🧠 Identify key factors influencing churn  
- 📊 Visualise important business KPIs  
- 💡 Support data-driven decision making  

---

## 🛠️ Tools Used
- 📊 Power BI  
- 📁 Microsoft Excel / CSV Dataset  
- 🧮 Data modelling and DAX measures  

---

## ⭐ Key Features
- 🎛️ Interactive dashboard with filters and slicers  
- 🧾 KPI cards showing:
  - 👥 Total Customers  
  - 📉 Churn Rate  
  - 🔵 Active vs 🔴 Churned Customers  
- 📈 Visual analysis of:
  - 💰 Monthly charges vs churn  
  - 📄 Contract type impact  
  - 👤 Customer demographics trends  

---

## 🔎 Key Insights
- 💸 Higher monthly charges may correlate with increased churn  
- 📑 Certain contract types show stronger customer retention  
- 📡 Customer engagement patterns help predict churn risk  

---

## 🚀 How to Use
1. 📂 Open the Power BI file (.pbix)  
2. 📥 Load the dataset if required  
3. 🎚️ Interact with slicers and filters to explore insights  
4. 📊 Review KPI cards and visual charts for patterns
